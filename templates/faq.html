{{ _("How does this work?") }}

{{ _("During placement of the waste bins, every site is registered in the database with its number and position. It gets a label telling the visitors to report full bins using a URL specific for every drop point. On the website, the visitor can report the drop point as needing attention by simply clicking a button. Optionally, the visitor may inform the waste collectors about the filling level of the drop point.") }}

{{ _("The other end of the web interface shows a list of drop points to visit ordered by priority: Bins reported a lot lately get a high priority which is even higher if the respective Bin experiences a lot or traffic and is reported multiple times. {{ _("What is the priority and how is it calculated?") }}

{{ _("Each Bin gets assigned a visit priority and the idea is that the waste collectors visit the Bins in order of their priority.") }}

{{ _("The priority is calculated by taking the time since the last visit as a base value relative to a base visit interval that determines how often a typical Bin should be visited. Each report of a Bin adds to the priority, where the severity of the report (the most severe is obviously \"overflowing\"). The latest report counts the most and earlier reports count less and less. The crate count of the Bin gives an additional bonus so that larger drop points get a slight increase in priority since a higher throughput is expected in such locations.") }}
